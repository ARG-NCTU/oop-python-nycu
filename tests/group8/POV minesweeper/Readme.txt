================================================================
MINESWEEPER 3D: SURVIVAL (3D 極限踩地雷)
================================================================

這是一個使用 Python 與 Ursina Engine 開發的 3D 第一人稱踩地雷遊戲。
這不僅僅是傳統的邏輯解謎，還結合了 FPS 移動機制、生存要素（躲避雷射）
與道具系統，讓踩地雷變得更加刺激！

[ 遊戲特色 ]
----------------------------------------------------------------
* 3D 第一人稱視角 (FPP)：身歷其境地走在雷區上。
* 生存模式 (Survival Mode)：除了掃雷，還必須躲避空中雷射並尋找補給。
* 硬派模式 (Hardcore Mode)：增加真實物理判定，踩到地雷也會引爆！
* 戰術視角 (TPP)：可切換至上帝視角觀察局勢。
* 流暢操作：支援衝刺、跳躍與穩定的碰撞判定。

[ 安裝與執行 ]
----------------------------------------------------------------
1. 系統需求：
   - Python 3.x
   - Ursina Engine

2. 安裝指令：
   pip install ursina

3. 啟動遊戲：
   python main.py

[ 操作說明 ]
----------------------------------------------------------------
W, A, S, D       : 移動角色
Space (空白鍵)   : 跳躍
Shift (按住)     : 衝刺 (加速)
滑鼠移動         : 轉動視角
滑鼠左鍵         : 挖掘 / 翻開方塊 (點到地雷即死)
滑鼠右鍵         : 插旗 / 標記 (防止誤點)
V 鍵             : 切換視角 (第一人稱 / 戰術視角)
Esc              : 暫停 / 選單

[ 遊戲模式設定 ]
----------------------------------------------------------------
在主選單中，您可以自由組合以下設定：

1. 難度選擇
   - ROOKIE (菜鳥)  : 10x10 網格，10 顆地雷
   - VETERAN (老手) : 15x15 網格，35 顆地雷
   - EXPERT (專家)  : 20x20 網格，70 顆地雷

2. 特殊模式 (可同時開啟)

   (A) 生存模式 (Survival Mode)
       - 玩家擁有 3 點生命值 (HP)。
       - 空中會隨機生成「雷射砲」攻擊玩家，請注意紅色警示與聚光燈。
       - 地圖會隨機生成道具：
         [紅色球體]: +HP (恢復生命)
         [青色鑽石]: SHIELD (護盾，可抵擋一次傷害)

   (B) 硬派模式 (Hardcore Mode)
       - 增加真實踩踏判定。
       - 如果移動時腳下「踩到」了未標記的地雷，會立即引爆導致任務失敗。
       - 請隨時留意腳下！

[ 遊戲規則 ]
----------------------------------------------------------------
1. 勝利條件：
   - 翻開地圖上所有「非地雷」的方塊。

2. 失敗條件：
   - 點擊（挖掘）到地雷。
   - 在硬派模式下踩到地雷。
   - 在生存模式下生命值歸零。

3. 提示：
   - 方塊上的數字代表周圍 9 宮格內的炸彈數量。
   - 第一次點擊有安全保護，絕對不會是地雷。

[ 檔案結構 ]
----------------------------------------------------------------
* main.py          : 遊戲主程式
* game_config.py   : 設定檔 (難度、顏色)
* map_manager.py   : 地圖生成與邏輯管理
* ui_manager.py    : UI 介面管理
* laser_system.py  : 雷射攻擊系統
* game_entities.py : 遊戲物件 (方塊、道具)
