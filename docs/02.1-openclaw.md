## How to Run

### Send message

Test if you can send a message in Slack
```bash
openclaw message send -t channel:#general -m "Hello from OpenClaw"
```

### TUI

Make sure you start with TUI. This may take a while first time to load things to memory

```bash
openclaw tui
```

Check the CPU/GPU used in a separated terminal
```bash
ollama ps
```

- You may see 55%/45% CPU/GPU for gemma4:e4b model.
